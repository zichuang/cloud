<b>回收站</b>

40.	设置二级回收站，防止用户误删文件。

一级回收站：用户回收站：用户删除文件后，文件进入用户回收站，用户回收站的文件，用户自己可以完全恢复。用户回收站的文件，只是对表字段state的值由1变为0，用户回收站的文件的自定义属性、文件的日志等信息，不会删除。所以用户回收站中的文件，可以完全恢复。

二级回收站：系统回收站：用户回收站的文件，只保留3天(系统可以设置保留天数)， 3天后，自动删除进入系统回收站。系统回收站只保留物理源文件，而文件的自定义属性、文件的日志、缩略图等信息，都会彻底删除。
用户回收站的保留天数，默认3天，系统设置可以进行设置这个天数，也可以设置为10天等
